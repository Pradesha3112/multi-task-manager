
# 🚀 Multi-Tenant Task Management System

A secure task management application supporting multiple organizations with role-based access control.

## ✨ Features

- 🏢 **Multi-Tenant**: Complete data isolation between organizations
- 👥 **Role-Based Access**: Admin and Member roles with different permissions
- 🔐 **JWT Authentication**: Secure login and registration
- 📝 **Task Management**: Create, read, update, and delete tasks
- 📋 **Audit Logging**: Track all actions with timestamps
- 🐳 **Docker Support**: Easy deployment with Docker Compose
- 🎨 **Modern UI**: Beautiful responsive interface

## 🛠 Tech Stack

- **Frontend**: React.js, React Router, Axios
- **Backend**: Node.js, Express.js
- **Database**: PostgreSQL
- **Auth**: JWT, bcrypt
- **DevOps**: Docker, Docker Compose

## 🚀 Quick Start

### With Docker (Recommended)
```bash
docker-compose up --build
```
##Manual Setup
bash
# Backend
```
cd backend && npm install && npm start
```
# Frontend
```
cd frontend && npm install && npm start
```
 Database Schema

organizations - Multi-tenant support

users - User accounts with roles

tasks - Task management

audit_logs - Activity tracking
